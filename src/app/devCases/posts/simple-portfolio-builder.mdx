## 项目概述

Once UI 是一个使用现代 Web 技术构建的开源设计系统，旨在为开发团队提供一套全面的可复用组件和设计 tokens。该系统通过模块化的架构和灵活的主题系统，帮助开发人员快速构建一致且美观的用户界面。

## 核心功能

- **丰富的组件库**: 提供包括按钮、表单、导航、布局等在内的 50+ 个 UI 组件
- **设计 Tokens**: 统一的颜色、字体、间距等设计规范，确保界面一致性
- **主题定制**: 支持深色/浅色模式，可轻松自定义主题色彩
- **响应式设计**: 所有组件均支持移动端、平板和桌面端的响应式布局
- **TypeScript 支持**: 完整的类型定义，提供更好的开发体验
- **无障碍访问**: 遵循 WCAG 2.1 AA 标准，确保所有用户都能使用

## 技术栈

- **React**: 用于构建组件
- **TypeScript**: 提供类型安全
- **CSS-in-JS**: 使用 styled-components 实现样式隔离
- **Storybook**: 用于组件文档和测试
- **Vite**: 快速的构建工具

## 使用场景

Once UI 适用于各种类型的 Web 应用开发，包括：

- **企业级应用**: 提供完整的企业 UI 解决方案
- **内容管理系统**: 丰富的表单和布局组件
- **数据可视化**: 强大的图表和数据展示组件
- **营销网站**: 现代化的视觉设计和交互效果
